# landing_page

## Introduction
This project is a responsive landing page built using only **HTML** and **CSS** as part of The Odin Project curriculum. The goal is to recreate a provided design mockup as accurately as possible while focusing on proper page structure, layout techniques, and clean styling practices.

By completing this project, I will have demonstrated my ability to:

- Structure web pages using semantic HTML5 elements
- Style and organize layouts using CSS
- Use Flexbox to position and align page elements
- Apply typography, spacing, colors, and visual hierarchy effectively
- Build multi-section landing pages from a design reference
- Write clean, readable, and maintainable code
- Organize a project and manage version control with Git and GitHub

This project represents an important step in strengthening my front-end development fundamentals and translating a static design into a fully functioning webpage using core web technologies.

---

## Features
- Responsive navigation header
- Hero section with call-to-action button
- Information cards section
- Testimonial/quote section
- Call-to-action banner
- Footer with copyright information
- Clean and modern layout built entirely with HTML and CSS
